# 릴리즈

## ver. 0.1 (2021-08-11)

### 주요 기능

* 검색기능
  - 작업 검색 : 전체검색, 정기점검, 기술지원, 장애, 신규설치
  - Column 검색 : 통합검색, 회사명, 부서명, 고객명, ..., 작업세부사항, 이슈사항, 엔지니어명
* PDF 기능
  - 작성된 리포트를 PDF 파일로 변환 및 다운로드
  - 고객 서명 하기(PDF 미리보기에서)
* 패스워드 변경 기능
  - 메뉴 -> 아바타 초상화 클릭  
  - 암호방식 변경(bcrypt)으로 모든 비번 초기화
  - (초기화 비번은 카톡으로 공지)
