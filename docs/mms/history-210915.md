# DCM 서비스

## 목적
* 영업 성과 증진
* 업무 효율 향상

## 목표
* 고객 맞춤형 정보 제공
* 업무 일원화

## 현재 서비스
* 리포트 게시판
* 리포트 PDF 변환
* 리포트 서명
* 리포트 공유

## 향후 고객 서비스
* 보고서 제공
  - 주기 : 월간/분기/반기/년간
  - 내용
    - 정기점검 데이터 그래프 : CPU, Memory, Disk
    - 주요 이슈 사항?
    - 점검 인원 OR 횟수 OR 시간
  - 양식 : 미정
  - 방법 : 미정
* 기술자료 제공
  - 내용 : 각 부서별? 또는 회사 차원?
  - 종류 : 문서, 동영상 등
  - 양식 : 디자인 필요
  - 방법 : 링크공유(카톡링크, 메일링크 등), PUSH 알림, PDF 문서 등

## 향후 사내 서비스 
* 기술자료 게시판
  - 내용 : 
* 업무 게시판  
  - 내용 : 리포트ID OR 기술자료ID OR 업무 내용
    - 리포트ID를 적으면 그 내용이 리스트 및 본문에 보이게 함
    - 영업부는 내용만 적으면 됨
    - BI에는 업무ID만 적으면 됨
* 업무내용 공유기능  
  - 카톡 OR 자체 PUSH 알림
